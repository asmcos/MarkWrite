Servers
    左侧为服务器列表（本地 / 远程 1 / 远程 2 等），右侧为当前选中服务器的一组配置。
    Workspace（只读显示）
    当前工作区路径仅供查看。要切换或修改工作区，请在主界面使用左侧 Files 面板或顶部 File > Open Workspace…。
    登录 / 注册

    使用 EventStore 密钥管理当前登录身份

    KEY LOGIN
    登录只支持两种方式：粘贴已有 ESEC，或输入邮箱生成新用户密钥。
    新用户生成只会填入输入框；只有在“用户信息”页点击保存到服务器时，才会真正保存身份。

    个人资料

    头像来自服务器已保存的资料（若已设置）；填写名称与简介后保存，将同步到 EventStore 个人主页

    身份密钥（只读）

    公钥与 ESEC 私钥由「登录 / 注册」页决定，不可在此修改；备份或迁移时请复制到安全位置。

    请勿泄露给他人；获得私钥即可代表您登录。
